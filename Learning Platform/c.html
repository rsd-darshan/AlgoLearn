Course Overview

This course covers various topics in C Programming, including basic syntax, data types, control structures, functions, pointers, arrays, and more. Each section provides detailed explanations and examples to help you understand and apply C programming concepts.

Introduction

C is a general-purpose programming language that is extremely popular, simple, and flexible to use. It is a structured programming language that is machine-independent and extensively used in various applications.

// Example: Hello World in C
#include <stdio.h>

int main() {
    printf("Hello, World!");
    return 0;
}

History of C

C was developed in the early 1970s by Dennis Ritchie at Bell Labs. It was designed to be compiled using a relatively straightforward compiler, to provide low-level access to memory, to provide language constructs that map efficiently to machine instructions, and to require minimal run-time support.

Features of C

  • Simple
  • Portable
  • Mid-level programming language
  • Structured programming language
  • Extensible
  • Rich library

Data Types

Data types in C specify the type of data that a variable can hold. There are several types of data types in C:

  • Derived Data Types: array, pointer, structure, union
  • Enumeration Data Type: enum
  • Void Data Type: void

Bitwise Operators

Bitwise operators perform bit-level operations on data. Common bitwise operators include AND (&), OR (|), XOR (^), NOT (~), left shift (<<), and right shift (>>).

// Example: Bitwise Operators
#include <stdio.h>

int main() {
int a = 5; // 0101 in binary
int b = 9; // 1001 in binary
printf("a & b = %d\n", a & b);  // 0001 = 1
printf("a | b = %d\n", a | b);  // 1101 = 13
printf("a ^ b = %d\n", a ^ b);  // 1100 = 12
printf("~a = %d\n", ~a);  // 1010 = -6
printf("a << 1 = %d\n", a << 1);  // 1010 = 10
printf("a >> 1 = %d\n", a >> 1);  // 0010 = 2
return 0;
}

Dynamic Memory Allocation

Dynamic memory allocation in C is performed using functions such as malloc(), calloc(), realloc(), and free(). These functions are used to allocate and deallocate memory at runtime.

// Example: malloc() and free()
#include <stdio.h>
#include <stdlib.h>

int main() {
int *ptr;
int n = 5;
ptr = (int*)malloc(n * sizeof(int));
if (ptr == NULL) {
printf("Memory allocation failed\n");
return 1;
}
for (int i = 0; i < n; i++) {
ptr[i] = i + 1;
}
for (int i = 0; i < n; i++) {
printf("%d ", ptr[i]);
}
free(ptr);
return 0;
}

// Example: calloc() and free()
#include <stdio.h>
#include <stdlib.h>

int main() {
int *ptr;
int n = 5;
ptr = (int*)calloc(n, sizeof(int));
if (ptr == NULL) {
printf("Memory allocation failed\n");
return 1;
}
for (int i = 0; i < n; i++) {
ptr[i] = i + 1;
}
for (int i = 0; i < n; i++) {
printf("%d ", ptr[i]);
}
free(ptr);
return 0;
}

// Example: realloc()
#include <stdio.h>
#include <stdlib.h>

int main() {

ptr[i] = i + 1;
}
ptr = (int*)realloc(ptr, 10 * sizeof(int));
if (ptr == NULL) {
printf("Memory allocation failed\n");
return 1;
}
for (int i = 5; i < 10; i++) {
ptr[i] = i + 1;
}
for (int i = 0; i < 10; i++) {
printf("%d ", ptr[i]);
}

return 0;
}

Difference between Structure and Union

The main difference between a structure and a union is in memory allocation. In a structure, each member has its own storage location, whereas in a union, all members share the same memory location.

  • Structure: Each member has its own memory location. The total memory required is the sum of the memory required by each member.
  • Union: All members share the same memory location. The total memory required is the memory required by the largest member.
// Example: Difference between Structure and Union
    #include <stdio.h>
    
    struct Person {
        char name[50];
        int age;
        float salary;
    };
    
    union Data {
        int i;
        float f;
        char str[20];
    };
    
    int main() {
        struct Person p;
        union Data d;
        printf("Size of structure: %ld\n", sizeof(p));
        printf("Size of union: %ld\n", sizeof(d));
        return 0;
    }

        ptr = (int*)malloc(n * sizeof(int));
        if (ptr == NULL) {
            printf("Memory allocation failed\n");
            return 1;
        }
        for (int i = 0; i < n; i++) {
            ptr[i] = i + 1;
        }
        for (int i = 0; i < n; i++) {
            printf("%d ", ptr[i]);
        }

Introduction to Preprocessor Directives

Preprocessor directives are lines included in the code of programs preceded by a hash sign (#). These lines are not program statements but directives for the preprocessor. Common preprocessor directives include #define, #include, #ifdef, #ifndef, and #endif.

// Example: #define Directive
    #include <stdio.h>
    
    #define PI 3.14
    
    int main() {
        printf("Value of PI: %f\n", PI);
        return 0;
    }
    
// Example: #include Directive
    #include <stdio.h>
    #include <stdlib.h>
    
    int main() {
        printf("Hello, World!\n");
        return 0;
    }

Conditional Compilation

Conditional compilation allows you to compile certain parts of your program depending on conditions you specify using preprocessor directives.

// Example: Conditional Compilation
    #include <stdio.h>
    
    #define DEBUG
    
    int main() {
    #ifdef DEBUG
        printf("Debug mode is enabled\n");
    #endif
        printf("Hello, World!\n");
        return 0;
    }

Macros

Macros are fragments of code that are given a name. Whenever the name is used, it is replaced by the contents of the macro. Macros can take arguments and be used to create inline functions.

// Example: Macro with Arguments
    #include <stdio.h>
    
    #define SQUARE(x) ((x) * (x))
    
    int main() {
        int num = 5;
        printf("Square of %d: %d\n", num, SQUARE(num));
        return 0;
    }
